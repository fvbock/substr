substr

A string search library and tool written in golang using the Boyer-Moore algorithm. The tool, named "sift", is like a binary grep.

Description

substr is written in golang. It provides both a library and command-line tool to search for a string or other sequence of bytes within a body of data. The body of data can be another string, an array of bytes, or data provided by an io.Reader (i.e., an input stream coming from a file, the network, etc.).

It uses the Boyer-Moore algorithm, which does far better than a sequential search. Portions of the data being searched through can be skipped altogether.

Library

You should be able to reference this library by adding this import to your code:

import "github.com/ivancich/substr"

The documentation generated by:

"go doc substr"

should be (barely) enough to get you started.

There are a number of entry functions depending on the form your data and your search key are in. Some functions just return the first match. Other functions provide all matches, by returning a channel that feeds substr.Result objects, each containing an offset or error.

sift -- a command-line tool

Sift is somewhat modeled after the grep tool, although it's not a replacement. The data you're looking for can be specified as a string or as a sequence of hexadecimal bytes. It will optionally recursively descend through directories.

Running:

sift -help

will provide basic usage documentation.
